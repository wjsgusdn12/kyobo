## 🧾 개요
교보문고 사이트의 주요 기능을 **Spring Boot + JPA + Thymeleaf** 기반으로 한 웹 프로젝트입니다.  
보안과 UI까지 고려해 완성도 높은 구조를 목표로 진행했습니다.

## 🗓️ 개발 기간
**2025.05.12 ~ 2025.06.03 (약 3주)**

## 🛠️ 기술 스택
- **언어**: Java, HTML/CSS, JavaScript, jQuery, Thymeleaf
- **툴**: STS, SQL Developer, Apache Tomcat, Ubuntu 22
- **기술**: Spring Boot, JPA, Spring Security, Ajax

## ✨ 주요 기능
- 🔍 검색 팝업창 구현 (페이징 + Ajax)
- 🧑‍💻 회원가입/로그인/정보수정 (Ajax 중복 체크)
- 🔐 Spring Security 기반 로그인 상태 관리
- 📄 메인화면 Paging 처리
- 🌐 서버 배포 (Linux Ubuntu)

## 👨‍💻 담당 역할
- 팝업 검색 기능 및 페이징 처리
- 로그인 흐름 + 보안 처리 구현
- 배포 및 환경 구성 전체

## ✅ 성과 및 학습
- Spring Boot 프로젝트 구조의 실무 적용 능력 향상
- Ajax + 보안 처리 + 페이징 등 실전 기능 다수 구현
- 보안 및 배포를 포함한 전반적 개발 역량 강화
